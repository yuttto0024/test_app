プラズマ理工学などの単語帳へ「計算付きカード」を追加するための簡単なガイド

目的
- 単語帳カードに計算過程を含める際に、MathJax が正しくレンダリングできるように表記を揃える。

カードフォーマット（例）
- cards 配列の各要素は次のフィールドを持ちます。
  - `q`: 問題（HTML を含めて良い）。インライン数式は `$...$`、表示数式は `$$...$$` を推奨。
  - `a`: 簡潔な解答（文字列）。数式は `$...$` を利用。
  - `calculation` (任意): 計算手順を HTML で書いた文字列。長い手順は `<div class="calculation">...</div>` のようにしておくとスクロール表示が効く。

例:
```
{
  q: "$E = h\\nu$ を使って光子のエネルギーを求めよ（\( \lambda = 500\ \mathrm{nm} \)）",
  a: "$4.0\\times10^{-19}\ \mathrm{J}$",
  calculation: `
    <div class="calculation">
      \[ E = \dfrac{hc}{\lambda} \]
      <p>数値を代入して...</p>
    </div>
  `
}
```

注意点
- このリポジトリの `app.js` には読み込み時に数式表記を標準化するヘルパー `sanitizeDeckMath` を追加済みです。`\( ... \)` や `\[ ... \]` などの表記を自動で `$...$` / `$$...$$` に変換しますが、可能なら最初から `$...$` を使ってください。
- `calculation` フィールド内は HTML を許容します。長い手順は `<div class="calculation">` で囲んでください。
- MathJax の再描画は `app.js` 側で行われます。カードを追加したらブラウザで開いて表示を確認してください。

追加時のチェックリスト
- [ ] 問題文の数式が `$...$` または `$$...$$` であること
- [ ] `calculation` を使う場合は `<div class="calculation">` を検討
- [ ] 保存後、ブラウザで MathJax が崩れていないか確認

何か質問があれば教えてください。ファイル追加のテンプレート作成や自動化スクリプトも作れます。
